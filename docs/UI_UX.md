# Gold ERP UI/UX

## Design Philosophy

- Minimal
- Modern
- Fast
- Mobile First
- Responsive
- Apple-level Experience

## Theme

- Light Mode
- Dark Mode

## Main Screens

- Login
- Dashboard
- Customers
- Products
- Inventory
- Purchase
- Sales
- Installments
- Repairs
- Reports
- Settings

## Rules

- Every screen must be editable.
- Fast navigation.
- Consistent design.
- Clear typography.
- Touch-friendly controls.

## Goal

Create a clean, modern and efficient ERP interface for professional jewelry stores.
