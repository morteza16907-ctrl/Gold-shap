# Gold ERP Database Design

## Database Engine

PostgreSQL

## Design Principles

- UUID as primary key
- Soft Delete
- Audit Log
- Created At
- Updated At
- Optimistic Locking

- Settings
- Users
- Roles
- Permissions

Highly scalable enterprise database.
